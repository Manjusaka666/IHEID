OVERVIEW
-----------------------------------------------------------------------
This folder contains the mapping between different country name formats
of the IMS, WHO and World bank datasets.


SOURCE
-------------------------------------------------------------------------
IMS Names:
/Data/Raw/MIDAS_2000-2013

WHO Names:
/Data/Raw/WHO_GBD

World Bank Names:
/Data/Raw/World_bank_GDP


DESCRIPTION
----------------------------------------------------------------------------

The directory has the following format

	ctry_xwalk.dta ----This is the final crosswalk between the different name formats

	README.txt    -----what you are reading right now!
